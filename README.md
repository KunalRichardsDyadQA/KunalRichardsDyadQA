## 🚀 What We Build

### 🧪 Dyad TestPro — AI-Powered Test Case Generator
> The flagship QA utility used by the entire Dyad QA team

- ⚡ **Instant test case generation** from JIRA tickets using Claude AI (Sonnet 4.6)
- 🔄 **3-pass pipeline** — baseline → gap detection → semantic coverage check
- 📊 **100% coverage tracking** per ticket with Rovo-style analysis  
- 🔗 **Direct Zephyr Scale import** — one click to push TCs to JIRA
- 👥 **Multi-user** with JWT auth, admin dashboard, and per-user cost tracking
- 📈 **CEO-ready reports** — daily/weekly/monthly CSV exports with USD/INR costs
